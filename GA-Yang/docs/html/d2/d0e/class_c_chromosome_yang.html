Detailed Description

CChromosomeYang is one element in GA population.

Role: Represents one solution (chromosome of bits). Is compounds of list of

Responsibilities:

  1. Knows to perform crossover with another solution
  2. Knows to perform mutation
  3. Knows to identify it is illegal solution and if so it knows how to fix itself
  4. Knows to calculation the Fitness value according to Evaluation criteria
  5. Knows how to build random and consistent Chromosome
