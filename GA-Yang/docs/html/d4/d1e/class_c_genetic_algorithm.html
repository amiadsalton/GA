bool CGeneticAlgorithm::Run ( )

Execute genetic algorithm.

It runs the algorithm for variant SNPs combination length, and then it analyzes the results in order to find the max combination length that has correlation with the disease. It calls for the function Run with combination length that varies from 0 to hard coded number (8).

bool false in case of failure.

bool CGeneticAlgorithm::RunAllOrders ( )

Execute genetic algorithm.

It runs the algorithm for variant SNPs combination length (order), and then it analyzes the results in order to find the max combination length that has correlation with the disease. It calls for the function Run with combination length that varies from 2 to user defined maximum order.
